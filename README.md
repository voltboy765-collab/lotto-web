# 🎰 로또 1등 시뮬레이터

매주 5장씩 로또를 구매하면 1등에 당첨될 때까지 얼마나 걸릴까요?

## 📱 특징

- ✅ **모바일 반응형** - 스마트폰에서도 완벽하게 작동
- ✅ **매주 구매 모드** - 현실적인 시뮬레이션 (매주 5장)
- ✅ **3단계 속도 조절** - 느리게/보통/빠르게
- ✅ **실시간 통계** - 주수, 구매액, 소요 시간
- ✅ **교육적 가치** - 로또 확률 체감

## 🚀 사용 방법

### 방법 1: 바로 실행 (가장 쉬움)

1. `index.html` 파일을 더블클릭
2. 브라우저에서 자동으로 열림
3. "시작하기" 버튼 클릭!

### 방법 2: VS Code에서 실행

1. VS Code로 폴더 열기
2. `index.html` 우클릭 → "Open with Live Server"
3. 또는 파일 탐색기에서 `index.html` 더블클릭

### 방법 3: GitHub Pages로 배포 (온라인 공개)

1. GitHub에 새 repository 생성
2. 파일 3개 업로드:
   - index.html
   - style.css
   - script.js
3. Settings → Pages → Source를 "main branch"로 설정
4. 몇 분 후 `https://your-username.github.io/repository-name` 접속

## 🎮 사용법

1. **시작하기** 버튼 클릭
2. 당첨번호가 랜덤으로 생성됨
3. 매주 5장씩 자동으로 구매 시뮬레이션
4. 속도 조절 가능 (🐢 느리게 / 🚶 보통 / 🚀 빠르게)
5. 일시정지/재개 가능
6. 당첨되면 결과 화면 표시!
